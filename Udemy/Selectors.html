Span
The span tag is an inline container used to mark up a part of a text, or a part of a document. The span tag is easily styled by CSS or manipulated with JavaScript using the class or id attribute. The span tag is much like the div element, but div is a block-level element and span is an inline element.

Division
A div is a block-level element.
The div tag defines a division or a section in an HTML document. The div tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript. The div tag is easily styled by using the class or id attribute. Any sort of content can be put inside the div tag!
The div tag is used in HTML to make divisions of content in the web page like (text, images, header, footer, navigation bar, etc).

Id
The id global attribute defines an identifier (ID) which must be unique in the whole document.The syntax for id is: write a hash character (#), followed by an id name.Its purpose is to identify the element when linking (using a fragment identifier), scripting, or styling (with CSS).
